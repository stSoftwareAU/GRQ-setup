# GRQ-setup

2. ssh-copy-id nigel@10.0.0.11
3. mkdir ~/src && cd ~/src
4. git clone git@github.com:stSoftwareAU/GRQ-setup.git

6. git clone git@github.com:stSoftwareAU/GRQ.git
7. vi GRQ/.option.txt
